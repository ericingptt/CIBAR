getUserMedia 需要 HTTPS 或 localhost；http://localhost:PORT／http://127.0.0.1:PORT 在瀏覽器認定裡本來就算「安全上下文」，不需要另外裝憑證。只有當你要從「另一台裝置」用區網 IP 連到這台電腦（例如手機瀏覽器連電腦的開發伺服器）時才需要真的 HTTPS，這種情況最簡單的做法是直接測試已部署的 GitHub Pages 網址（本來就是 HTTPS）；如果一定要本機測，可以用 mkcert 產生受信任憑證後換一個支援 HTTPS 的簡易伺服器（例如 npx http-server -S -C cert.pem -K key.pem）。
